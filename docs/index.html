About

Valuelabs is a project focused on capital management using machine learning.

The systems are currently in development and can be seen been in version alpha. We have a great interest in Open Source and will therefore be sharing most of the source code in the hope that more companies and developers will do the same. Although, for obvious reasons, the secret sauce will be kept secret.

Frequent updates will be made to the graph showing the net worth of the project and the blog entries. So stay tuned for more updates!

2020-10-18

Website launch!

After hours of work, the first version of the website has now been launched.

Technical information:
The website is made using python and the flask library. Since the website's content is static, the choise to host it using github pages was beneficial in many ways. The security implications of hosting a website are minimized, the hosting costs are none and it is way easier to deploy. After the website is finished, using Frozen-Flask it is frozen and pushed to the github repository.
